Welcome to the Book Management API

  • GET /books
  • POST /books
  • GET /books/<book_id>
  • PUT /books/<book_id>
  • DELETE /books/<book_id>

Authentication

All endpoints require HTTP Basic Auth

  • Username: admin
  • Password: password

Examples of curl Requests


        curl -X GET https://books-management-api-6389.onrender.com/books -u admin:password

GET /books (List all books)

        curl -X GET https://books-management-api-6389.onrender.com/books -u admin:password

POST /books (Add a new book)

        curl -X POST https://books-management-api-6389.onrender.com/books -u admin:password -H "Content-Type: application/json" -d '{"title": "New Book", "author": "John Doe", "published_date": 2025, "isbn": "1234567890123"}'

GET /books/<book_id> (Get a book by ID)

        curl -X GET https://books-management-api-6389.onrender.com/books/1 -u admin:password

PUT /books/<book_id> (Update a book by ID)

        curl -X PUT https://books-management-api-6389.onrender.com/books/1 -u admin:password -H "Content-Type: application/json" -d '{"title": "Updated Book", "author": "Jane Doe", "published_date": 2025, "isbn": "1234567890123"}'

DELETE /books/<book_id> (Delete a book by ID)

        curl -X DELETE https://books-management-api-6389.onrender.com/books/1 -u admin:password
